## 2. 빌드 시 환경변수 및 주의사항

1.  빌드 시에 vm-option으로 -Djasypt.password=${password} 필요
2.  yml 작성시 password나 외부 서비스 사용 key를 jasypt로 인코딩 후 작성.
3.  농협 오픈 플랫폼 API 사용 불가
